# Smart Library Manager 📚

a robust, desktop-based Library Management System developed entirely in Java. Designed with a clean, modern Swing GUI, the system ensures optimal performance by implementing advanced algorithmic data structures beneath the surface.

## ✨ Key Features
* **Algorithmic Efficiency:** Uses **AVL Trees** for lightning-fast book indexing and retrieval, and **Min-Heaps** for managing priority-based student waitlists.
* **Comprehensive Cataloging:** Add, search, and manage library books with real-time tracking of total and available copies.
* **Student Registry:** Register students and track their academic status (Active/Graduated) to determine waitlist priorities.
* **Smart Borrowing System:** Secure borrow/return transactions with automated waitlist processing and fine calculations.
* **System Analytics:** Generate full overview reports identifying the most popular books, active borrows, and overdue alerts.
* **Modern UI/UX:** A flat, elegant interface styled with custom colors and typography for a seamless user experience.

## 🛠️ Tech Stack
* **Language:** Java (JDK 11+)
* **GUI Framework:** Java Swing / AWT
* **Core Data Structures:** AVL Tree, Min-Heap, Hash Maps
